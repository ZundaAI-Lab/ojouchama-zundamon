お嬢ちゃまずんだもんと夢みる豆の王国 完成版プロトタイプ
====================================================

HTML / CSS / JavaScript (ES Modules) で構成した、メルヘン横スクロール探索アクションです。
Canvas を中心にゲーム画面を描画し、UI は DOM ビューとして重ねて表示します。

■ 操作
初期キー設定は以下の通りです。ゲーム内のキーコンフィグから変更できます。

- ← → / A D : 左右移動
- ↑ / W : 上方向
- ↓ / S : 下方向
- X / Space : ジャンプ
- Z / J : 豆の魔法
- C / K : おじぎ
- V / L : ティータイム
- Shift / H : なのちゃん
- P / Esc : ポーズ / 戻る
- Enter / Space / Z : 決定 / 会話送り
- Esc : キャンセル

タッチ操作が有効な場合は、画面左側の方向パッドと右側の「茶」「礼」「なの」「魔法」「飛」ボタンで操作します。
タッチ操作の有効/無効、左右配置、サイズ、透明度はゲーム内設定から変更できます。

■ 実行方法
ES Modules を使用しているため、ローカルファイルを直接開かず、ローカルサーバー経由で起動してください。

例:
  cd ojouchama_zundamon_game
  python -m http.server 8000

その後、ブラウザで以下を開いてください。
  http://localhost:8000/

Windowsでは同梱の run_local_server.bat でも起動できます。

■ 構成
- index.html  : 起動HTML
- styles      : CSS読み込み口、共通レイアウト、HUDなどの画面装飾
- assets      : 画像アセット
- src/main.js : アプリ起動エントリ
- src/core    : アプリ本体、ゲームループ、カメラ、シーン管理
- src/scenes  : Title / Opening / Garden / Option / KeyConfig / TouchControl / Shop / Stage / Result / Debug
- src/stage   : ステージ進行、生成、エリア管理、ギミック、衝突後処理、クリア処理、ポーズ/リトライなどの実行フロー
- src/systems : 入力、保存、アセット、物理、衝突判定
- src/render  : Canvas描画
- src/audio   : BGM/SEとWebAudio制御
- src/actors  : Actor基底、プレイヤー、なのちゃん、弾、住民、ボス、アイテム、ゴール
- src/data    : ワールド、ステージ、アセット、住民、アイテム、ボス、足場定義
- src/config  : 表示、入力、プレイヤー、ワールド、強化、難易度、チュートリアルの定義
- src/ui      : HUD、会話ウィンドウ、DOMビュー、メニュー操作
- src/debug   : デバッグ表示とデバッグテスト
- src/utils   : 汎用関数

■ 備考
仕様書全体に対して、ブラウザ上で最後まで遊べる一通りのゲームフローを実装しています。
アニメーションはスプライトシート連番ではなく、生成したポーズ画像の状態切り替えで表現しています。
通常ステージは `src/data/stages/<world>/area_1.js`、`area_2.js`、`area_3.js`、`boss.js` に分割して定義しています。
デバッグ用の `switch_test_lab` は、通常のワールド進行とは別の検証用ステージです。

## 2026-05-03 現行構成メモ

この版では、責務別ディレクトリを現行の唯一の構成として使用します。音声処理は `src/audio/`、描画処理は `src/render/`、ステージ実行処理は `src/stage/`、種類別Actorは `src/actors/player/`・`src/actors/nano/`・`src/actors/resident/`・`src/actors/boss/`・`src/actors/item/` に配置しています。

- `src/stage/`: ステージ実行中の進行、生成、エリア管理、ギミック、衝突後処理、クリア処理、リトライ、ポーズ、報酬処理。
- `src/stage/runtime/`: StageRuntime本体から分離した、HUD表示、ポーズ、プレイヤー行動、弾、更新フロー、リトライ、初期化処理。
- `src/stage/balloonRide/`: 風船ライド専用の状態、接触解決、風船在庫、演出、結果処理。
- `src/actors/player/`, `src/actors/nano/`, `src/actors/resident/`, `src/actors/boss/`, `src/actors/item/`: Actorの種類別責務。
- `src/data/stages/<world>/`: 各ワールドの3エリア＋ボスステージ定義。
- `src/data/stages/routes/`: ワールド別ルートの集約。
- `src/data/*Defs.js`: 住民、アイテム、ボス、足場などの静的定義。
- `src/render/`: Canvas描画系。
- `src/audio/`: 音声系。
- `src/config/`: 表示、入力、プレイヤー、ワールド、強化、難易度、チュートリアルの定義。
- `src/ui/views/`: DOMビュー。
- `styles/index.css`: CSS読み込み口。画面別CSSと部品CSSを現在の表示順に沿って読み込みます。

各JS/CSS/HTML/TXT/BATには、責務と更新ルールのコメントを追加しています。
